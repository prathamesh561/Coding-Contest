DESCRIPTION

Given mark of a student tell how the student is feeling.

        if marks are betwenn 75 student is very happy.
        if marks are bewtween 50 and 74 student is happy.
        if marks are bewteen 25 and 49 student is sad.
        if marks are less than 25 student is very sad.

Sample

Input 

The first line contains an integer N denoting the number of students
        Then N lines follow with marks of a student in each line

Output 

For each student print how the student is feeling
