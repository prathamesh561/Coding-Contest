Name with Age

International Coding Contest > Scoreboard > Problem B

DESCRIPTION

In first line, given a name as input. In second line, given the age as input.

        Print it this way: {Name} is {age} years old.
        
        For example if name is Rama and age is 5 then print Rama is 5 years old.

Sample

Input

Rama

5

Output

Rama is 5 years old.
